# Sitios Web — Negocios Locales

Páginas web desarrolladas para negocios en Monterrey, México. Cada sitio incluye animaciones de entrada, diseño responsivo mobile-first, integración de WhatsApp, y está optimizado para SEO y rendimiento.

**Tech:** HTML5 · CSS3 · JavaScript · Google Fonts · Intersection Observer API
